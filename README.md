# AI Document Intelligence Platform

This project is an AI-powered document intelligence system that enables users to upload documents (PDFs/images), extract text using OCR, and interact with documents using NLP-based question answering.

## Initial Setup

This repository contains the initial project structure for the AI Document Intelligence Platform.

##  Planned Features

- OCR-based text extraction from PDFs/images
- BERT/SBERT embeddings for semantic understanding
- FAISS-based vector search
- RAG (Retrieval-Augmented Generation) pipeline
- FastAPI backend for APIs
- Dockerized deployment setup
- Logging and modular architecture
